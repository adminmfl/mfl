# V2.5 AI League Manager — Demo Flow

## Overview

The AI League Manager is a host-facing intelligence layer that helps league hosts manage their leagues more effectively. It provides daily health digests, AI-drafted communications, intervention alerts for at-risk members, and a challenge template library with automated communication schedules.

## Access

- Navigate to any active league as a **Host** or **Governor**
- Click **"AI Manager"** in the sidebar (under Host Actions)
- URL: `/leagues/[leagueId]/ai-manager`

---

## Tab 1: Digest (Daily Health Check)

**What it shows**: Prioritized insights about your league's health today.

### How to populate data:
1. Click **"Run Scan Now"** button (top right)
2. The scan evaluates 9 rules against your league's current state:
   - Participation drop (league-wide < 70%)
   - Per-team participation drop (< 50%)
   - Inactive members (3+ days)
   - Broken streaks
   - Team imbalance (40%+ point gap)
   - Challenge ending soon (2 days or less)
   - Run rate anomalies
   - Final week/last day alerts
   - Week 1 milestone

### Actions:
- **Mark as read** (eye icon)
- **Dismiss** (hide icon)

---

## Tab 2: Drafts (AI-Generated Communications)

**What it shows**: AI-drafted messages ready for host review, edit, and send.

### How drafts are generated:
1. From **Alerts tab** → click "Generate Message" on any intervention
2. From **Challenge deploy** → scheduled communications auto-generate drafts
3. Drafts use Mistral AI to create contextual, personalized messages

### Actions:
- **Send** → delivers the message through the league messaging system
- **Edit** → modify the AI-generated text before sending
- **Dismiss** → mark as not needed
- **Delete** → permanently remove

### Draft types:
- `nudge` — gentle reminder to inactive player
- `team_nudge` — team motivation message
- `announcement` — league-wide broadcast
- `intervention` — personal check-in for at-risk member
- `challenge_hype` — challenge kickoff message
- `challenge_results` — challenge results celebration

---

## Tab 3: Alerts (Member Interventions)

**What it shows**: At-risk members detected by the scan, ranked by severity.

### Severity levels:
- **Critical** (red) — 5+ days inactive, dropout risk
- **High** (orange) — 3-4 days inactive
- **Medium** (yellow) — streak broken, rest days exhausted
- **Low** (blue) — declining run rate

### Trigger types:
- `inactivity_5d` — 5+ days no activity
- `inactivity_3d` — 3-4 days no activity
- `streak_broken` — was active, missed yesterday
- `rr_declining` — run rate trending down
- `rest_days_exhausted` — used all rest days
- `dropout_risk` — inactive + most rest days used

### Actions:
- **Generate Message** → creates an AI draft (appears in Drafts tab)
- **Dismiss** → mark as handled

---

## Tab 4: Library (Challenge Templates)

**What it shows**: Pre-built challenge templates with automated communication schedules.

### Available templates (10 seed templates):
1. Step It Up (7 days, steps)
2. Consistency King (7 days, consistency)
3. Early Bird (5 days, time-based)
4. Team Spirit Week (7 days, team)
5. Distance Dash (7 days, distance)
6. Weekend Warrior (2 days, weekend)
7. Mindful Minutes (7 days, wellness)
8. Photo Proof (5 days, proof)
9. Power Hour (5 days, duration)
10. League Finale (3 days, finale)

### Deploy flow:
1. Click **"Deploy"** on any template
2. Set start date and optionally customize the name
3. Review the communication schedule (e.g., Day 0: hype, Day 3: check-in, Day 6: results)
4. Click **"Deploy"** → creates the challenge + schedules AI communications
5. Scheduled comms auto-generate drafts on the scheduled dates (via daily cron)

---

## Daily Cron Job

- **Schedule**: Runs at 3:00 AM UTC daily
- **Path**: `/api/cron/ai-daily-scan`
- **What it does**:
  1. Finds all active leagues
  2. Runs digest evaluation + intervention detection
  3. Processes scheduled challenge communications
  4. Generates drafts for any communications due today

---

## End-to-End Demo Sequence

1. Open AI Manager as host
2. Click **"Run Scan Now"** → digest and alerts populate
3. Review **Digest tab** → see league health insights
4. Switch to **Alerts tab** → see at-risk members
5. Click **"Generate Message"** on a critical alert → draft is created
6. Switch to **Drafts tab** → see the generated draft
7. Click **"Edit"** → customize the message → **"Save"**
8. Click **"Send"** → message delivered to team chat
9. Switch to **Library tab** → browse challenge templates
10. Click **"Deploy"** on "Step It Up" → set start date → deploy
11. Show the scheduled communications that will auto-generate

## API Endpoints Summary

| Method | Endpoint | Purpose |
|--------|----------|---------|
| GET | `/api/leagues/[id]/host-digest?date=YYYY-MM-DD` | Fetch digest items |
| PATCH | `/api/leagues/[id]/host-digest` | Mark read/dismissed |
| GET | `/api/leagues/[id]/drafts?status=pending` | List drafts |
| POST | `/api/leagues/[id]/drafts` | Generate new draft |
| PATCH | `/api/leagues/[id]/drafts/[draftId]` | Edit/dismiss draft |
| DELETE | `/api/leagues/[id]/drafts/[draftId]` | Delete draft |
| POST | `/api/leagues/[id]/drafts/[draftId]/send` | Send draft as message |
| GET | `/api/leagues/[id]/interventions?status=pending` | List interventions |
| PATCH | `/api/leagues/[id]/interventions` | Update status |
| POST | `/api/leagues/[id]/interventions/[id]/draft` | Generate draft from alert |
| GET | `/api/leagues/[id]/challenge-templates` | List templates |
| POST | `/api/leagues/[id]/challenge-deploy` | Deploy template |
| POST | `/api/leagues/[id]/ai-scan` | Manual scan trigger |
| GET/POST | `/api/cron/ai-daily-scan` | Daily cron job |
